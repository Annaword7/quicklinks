# Dev Link Manager - Расширение для Chrome

Расширение для управления ссылками с переменными для разработчиков, продакт-менеджеров и проджект-менеджеров.

## Возможности

### 🔗 Шаблоны ссылок с инлайн-вводом
- Создавайте шаблоны URL с переменными в фигурных скобках: `{hotelId}`, `{rateId}` и т.д.
- **Мгновенный ввод**: Вводите параметры прямо в URL без модальных окон
- Организуйте шаблоны по категориям
- Фильтруйте шаблоны по категориям

### 📋 Буфер обмена (15 последних значений)
- Автоматическое отслеживание последних 15 скопированных значений
- Клик по значению для автоматической вставки в активное поле
- Идеально для быстрого копирования ID и вставки

### ⭐ Избранное
- Сохраняйте часто используемые ссылки с уже подставленными параметрами
- Быстрый доступ к готовым ссылкам одним кликом

### 📊 История переходов
- Автоматическое сохранение всех открытых ссылок
- Просмотр истории с временными метками
- Очистка истории одним кликом

### 🪟 Отдельное окно
- Расширение открывается в отдельном окне, а не в popup
- Работайте без потери данных при переключении между вкладками

## Установка

1. Скачайте папку с расширением
2. Откройте Chrome и перейдите на `chrome://extensions/`
3. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
4. Нажмите "Загрузить распакованное расширение" (Load unpacked)
5. Выберите папку `link-manager-extension`
6. Расширение установлено! Нажмите на иконку в панели инструментов

## Использование

### Создание шаблона

1. Откройте расширение (откроется отдельное окно)
2. В разделе "Шаблоны" заполните форму:
   - **Название**: Например, "Hotel Info Page"
   - **URL**: `https://example.com/hotels/{hotelId}/info`
   - **Категория**: Например, "Hotels" (опционально)
3. Нажмите "Добавить"

### Примеры шаблонов

```
Название: Hotel Info
URL: https://example.com/hotels/{hotelId}/info
Категория: Hotels

Название: Rate Plan Edit
URL: https://example.com/hotels/{hotelId}/rateplan/{rateId}/edit
Категория: Rates

Название: GitHub Issue
URL: https://github.com/{owner}/{repo}/issues/{issueNumber}
Категория: GitHub

Название: JIRA Task
URL: https://yourcompany.atlassian.net/browse/{taskId}
Категория: JIRA
```

### Быстрый workflow

1. **Скопируйте нужные ID** в других вкладках
2. **Откройте расширение** - все скопированные значения появятся в левой панели
3. **Кликните в поле ввода** шаблона
4. **Нажмите на значение** в буфере обмена - оно автоматически вставится
5. **Enter или кнопка "Открыть"** - ссылка откроется

### Работа с шаблонами

Каждый шаблон отображается в виде:
```
https://example.com/ [hotelId] /info/ [rateId] /edit  [Открыть →]
```

- Вводите значения прямо в поля
- Enter переходит к следующему полю
- Enter на последнем поле открывает ссылку
- Кнопка "⭐ В избранное" сохраняет готовую ссылку

### Работа с буфером обмена

1. Скопируйте любое значение (Ctrl+C / Cmd+C)
2. Оно автоматически появится в левой панели
3. Кликните по значению чтобы:
   - Скопировать его обратно в буфер
   - Автоматически вставить в активное поле ввода

### Работа с избранным

1. Перейдите на вкладку "Избранное"
2. Нажмите "Открыть" для быстрого перехода
3. Используйте "Удалить" для очистки ненужных ссылок

### Просмотр истории

1. Перейдите на вкладку "История"
2. Просмотрите все ваши переходы с временными метками
3. Используйте "Очистить историю" при необходимости

## Горячие клавиши

- **Enter** в полях добавления шаблона - добавить шаблон
- **Enter** в полях ввода параметров - перейти к следующему полю
- **Enter** на последнем параметре - открыть ссылку
- **Click** на поле ввода - автоматический выбор текста
- **Click** на значение в буфере - вставить в активное поле

## Технические детали

### Хранение данных
Расширение использует `chrome.storage.local` для сохранения:
- Шаблонов ссылок
- Избранных ссылок
- Истории переходов (последние 50 записей)
- Буфера обмена (последние 15 значений)

### Синтаксис переменных
Используйте фигурные скобки для определения переменных в URL:
- `{variableName}` - будет заменено на введенное значение
- Допустимы любые символы в имени переменной
- Можно использовать несколько переменных в одном URL

### Буфер обмена
- Проверяется каждые 2 секунды когда окно в фокусе
- Хранит последние 15 уникальных значений
- Автоматически удаляет дубликаты
- Требует разрешение `clipboardRead` в manifest.json
