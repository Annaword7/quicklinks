# Руководство по публикации в Chrome Web Store

## Подготовка к публикации

### 1. Проверьте расширение локально

1. Откройте `chrome://extensions/`
2. Включите "Режим разработчика"
3. Загрузите распакованное расширение
4. Протестируйте все функции

### 2. Подготовьте материалы для публикации

#### Обязательные материалы:

**Иконки** (уже созданы в папке `icons/`):
- 16x16 пикселей
- 48x48 пикселей
- 128x128 пикселей

**Скриншоты** (нужно создать):
- Минимум 1, максимум 5 скриншотов
- Размер: 1280x800 или 640x400 пикселей
- Формат: PNG или JPG
- Покажите основные функции расширения

**Promotional Image** (опционально, но рекомендуется):
- Размер: 440x280 пикселей
- Формат: PNG или JPG

#### Текстовые материалы:

**Краткое описание** (до 132 символов):
```
Управление ссылками с переменными для разработчиков и менеджеров. Быстрый доступ к часто используемым URL.
```

**Подробное описание**:
```
Dev Link Manager - мощный инструмент для управления ссылками с параметрами.

✨ ОСНОВНЫЕ ВОЗМОЖНОСТИ:

🔗 Шаблоны ссылок
• Создавайте URL-шаблоны с переменными
• Организуйте по категориям
• Перетаскивайте для сортировки

⭐ Избранное
• Сохраняйте готовые ссылки
• Быстрый доступ одним кликом

📊 История
• Автоматическое отслеживание переходов
• Удобный просмотр с датами

🎯 ИДЕАЛЬНО ДЛЯ:
• Разработчиков
• Продакт-менеджеров
• Проджект-менеджеров
• QA-инженеров

📝 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:
• Быстрый доступ к админ-панелям с разными ID
• Переходы на страницы GitHub/JIRA
• Работа с AWS консолью
• Навигация по внутренним системам

Все данные хранятся локально в браузере.
```

### 3. Создайте Privacy Policy

Создайте простую страницу с политикой конфиденциальности:

```
Privacy Policy for Dev Link Manager

Last updated: [Текущая дата]

This extension does not collect, store, or transmit any personal data.

All data (templates, favorites, history) is stored locally in your browser using Chrome's storage API.

No analytics or tracking tools are used.

Contact: [Ваш email]
```

Разместите её на:
- GitHub Pages
- Google Sites
- Вашем личном сайте

### 4. Регистрация в Chrome Web Store

1. Перейдите на https://chrome.google.com/webstore/devconsole/
2. Войдите с Google аккаунтом
3. Оплатите одноразовый регистрационный взнос $5 (если ещё не оплачивали)

### 5. Публикация расширения

1. Нажмите "New Item"
2. Загрузите ZIP-файл расширения
3. Заполните всю информацию:
   - **Store listing**:
     - Detailed description
     - Screenshots
     - Promotional images
     - Category: Productivity
     - Language: Russian (или другие)
   
   - **Privacy**:
     - Single purpose description
     - Permission justifications
     - Privacy policy URL
   
   - **Pricing & Distribution**:
     - Free
     - Select countries (или Worldwide)
     - Visibility: Public

4. Нажмите "Submit for review"

### 6. Процесс ревью

- **Время проверки**: обычно 1-3 дня
- **Возможные замечания**:
  - Недостаточно описания разрешений
  - Нужна более подробная privacy policy
  - Требуются изменения в manifest

### 7. После публикации

**Обновления**:
1. Увеличьте version в manifest.json
2. Создайте новый ZIP
3. Загрузите через Dashboard
4. Опишите изменения
5. Submit for review

**Мониторинг**:
- Отслеживайте отзывы пользователей
- Проверяйте статистику установок
- Отвечайте на вопросы

## Checklist перед публикацией

- [ ] Расширение работает корректно
- [ ] Все функции протестированы
- [ ] Создано минимум 3 скриншота
- [ ] Написана privacy policy и размещена онлайн
- [ ] Заполнены все описания
- [ ] Выбраны правильные разрешения в manifest
- [ ] Version установлена на 1.0.0
- [ ] ZIP-архив создан и протестирован
- [ ] Оплачен регистрационный взнос $5

## Советы для успешной публикации

1. **Качественные скриншоты**: Покажите все основные функции
2. **Чёткое описание**: Объясните, зачем нужно расширение
3. **Keywords**: Используйте релевантные ключевые слова
4. **Privacy Policy**: Даже если не собираете данные, она обязательна
5. **Обоснование разрешений**: Объясните, зачем нужны storage и tabs
6. **Регулярные обновления**: Поддерживайте расширение актуальным

Удачи с публикацией! 🚀
